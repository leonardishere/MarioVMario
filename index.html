Mario V Mario

Mario V Mario is a machine learning project that got pushed to the side and may be revisited in the future. At the moment, it serves as a good example of using JavaScript and JQuery to create an interactive page.

Instructions: This game is meant to just be watched. If you would like to play, the WASD keys control Luigi and the R key respawns. On death, a clone of you will be spawned.

This game is also a (bad) example of machine learning. The marios are controlled using neural networks and are trained via genetic algorithms. Clones are trained via gradient descent of your moves.
